# FastAPI Standard Backend

This is the backend for the `fastapi-standard` template project, built with FastAPI, SQLModel, Alembic, slowapi, and pytest.

## Getting Started

1. Set up dependencies:
   ```bash
   uv sync
   ```

2. Run development server:
   ```bash
   uv run fastapi dev app/main.py --port 8000
   ```

3. Run tests:
   ```bash
   uv run pytest
   ```
